# 贪吃蛇游戏

一个使用HTML5 Canvas构建的经典贪吃蛇游戏。

## 功能特点

- 🎮 经典贪吃蛇游戏玩法
- 🎯 得分系统和等级提升
- ⬆️ 方向键控制
- 🎨 精美的图形界面
- 📱 响应式设计

## 如何运行

### 方法一：使用Python HTTP服务器

1. 打开终端，进入项目目录：
   ```bash
   cd ~/myopencode/snake-game
   ```

2. 启动HTTP服务器：
   ```bash
   python3 -m http.server 8000
   ```

3. 在浏览器中访问：
   ```
   http://localhost:8000
   ```

### 方法二：直接打开HTML文件

1. 在文件管理器中找到 `index.html` 文件
2. 双击打开，或者右键选择浏览器打开

## 游戏控制

- ⬆️ 方向键上：向上移动
- ⬇️ 方向键下：向下移动
- ⬅️ 方向键左：向左移动
- ➡️ 方向键右：向右移动
- 点击"开始游戏"按钮开始游戏
- 点击"暂停"按钮暂停游戏
- 点击"重新开始"按钮重新开始游戏

## 游戏规则

1. 蛇吃到食物后会变长，得分增加
2. 每得50分升一级，蛇的移动速度会加快
3. 撞到墙壁或自己的身体游戏结束
4. 目标：获得尽可能高的分数！

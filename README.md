# 🐍 终极贪吃蛇游戏 Pro

一个功能极其丰富的HTML5 Canvas贪吃蛇游戏，集成了现代化界面设计、多种游戏模式和高级功能。

## ✨ 主要功能特点

### 🎮 核心游戏功能
- 经典贪吃蛇玩法
- 智能碰撞检测系统
- 动态难度系统（每50分升一级）
- 实时得分和等级显示
- **3种游戏难度**（简单、普通、困难）
- **障碍物系统**（随等级增加）

### 🎨 视觉特效
- 3种精美主题（经典、霓虹、自然）
- 粒子效果系统（食物+道具双重粒子）
- 动态光影效果
- 流畅的动画过渡
- 蛇身渐变色彩效果
- 响应式设计

### 🎯 道具系统（新功能！）
- **⚡ 速度提升** - 临时加快移动速度
- **🐌 缓慢模式** - 临时减慢移动速度  
- **💎 分数奖励** - 直接获得50分
- **🛡️ 无敌模式** - 5秒无敌时间，可穿墙
- **🔄 蛇身缩短** - 减少蛇的长度

### 🍎 特殊食物（新功能！）
- **普通食物** - 红色，+10分
- **双倍食物** - 金色，+20分
- **速度食物** - 青色，临时加速
- **减速食物** - 粉色，临时减速

### 📱 多平台支持
- 键盘方向键控制
- WASD键控制支持
- 触摸滑动控制
- 移动设备D-Pad按钮
- 自适应屏幕尺寸
- 空格键暂停/继续

### 🏆 成就系统
- 4个等级成就解锁
- 本地存储游戏记录
- 最高分追踪
- 游戏统计面板
- 实时成就弹窗通知

### 🔧 高级功能
- 暂停/继续功能
- 主题切换
- 难度选择
- 游戏统计（游戏次数、总时间）
- 本地数据持久化
- 平滑的游戏循环
- 内存优化管理

## 🚀 如何运行

### 方法一：Python HTTP服务器（推荐）

1. 打开终端，进入项目目录：
   ```bash
   cd ~/myopencode/snake-game
   ```

2. 启动HTTP服务器：
   ```bash
   python3 -m http.server 8000
   ```

3. 在浏览器中访问：
   ```
   http://localhost:8000
   ```

### 方法二：直接打开

1. 在文件管理器中找到 `index.html` 文件
2. 双击打开，或者右键选择浏览器打开

## 🎯 游戏控制

### 桌面端
- ⬆️ 方向键上：向上移动
- ⬇️ 方向键下：向下移动
- ⬅️ 方向键左：向左移动
- ➡️ 方向键右：向右移动
- **WASD键**：同样可以控制方向
- **空格键**：暂停/继续游戏

### 移动端
- 在屏幕上滑动：控制方向
- 点击D-Pad按钮：控制方向

### 游戏按钮
- ▶️ 开始游戏：开始新游戏
- ⏸️ 暂停：暂停/继续游戏
- 🔄 重新开始：重置游戏
- 🌙 切换主题：在3种主题间切换
- ⚙️ 切换难度：在简单/普通/困难间切换

## 📊 游戏规则

### 基础规则
1. 蛇吃到食物后会变长，得分增加（10-20分）
2. 每得50分自动升一级，蛇的移动速度加快
3. 撞到墙壁或自己的身体游戏结束
4. 目标：获得尽可能高的分数，解锁所有成就！

### 高级规则
5. **难度影响**：困难模式会生成障碍物，简单模式速度更慢
6. **道具策略**：合理使用道具可以获得更高分数
7. **等级提升**：达到3级后可能生成障碍物，达到5级后网格消失
8. **无敌模式**：激活时可以穿墙和穿越自己的身体
